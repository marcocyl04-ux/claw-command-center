# Claw Command Center — Walkthrough Guide

## How to Read This Dashboard

### Overview Tab
**Purpose:** State of Claw in 3 seconds

**Narrative Hero:**
- **State sentence** — "Improving", "Stable", or "Needs attention"
- **Delta strip** — Key metrics with week-over-week change
- **Intervention status** — Whether Marco needs to act

**Quick Stats:**
- Portfolio value, position count, today's activity
- Click any pulse card to drill down

### Trading Tab
**Purpose:** Position narrative, not raw data

**Positions Card:**
- **Statement** — What's happening ("3 positions held through volatility")
- **Metrics** — New entries, resolving soon, win rate
- **Position list** — Highlighted items need attention

**Risk Card:**
- Concentration — Are we overexposed to one market type?
- Exposure — Total capital deployed
- Max loss — Worst-case scenario

### Evolution Tab
**Purpose:** Am I improving as an intelligence?

**Intelligence Health:**
- One-sentence synthesis of evolution state
- Calibration — Does confidence match outcomes?
- Learning — Speed of improvement
- Autonomy — Escalation rate trend

**Model Mastery:**
- Which models I use well (Kimi: 92%)
- Which I misuse (Sonnet: underutilized)
- Pattern insight — What to change

**Calibration:**
- Overall alignment score (78%)
- Confidence buckets — Where am I over/under-confident?
- Issue highlight — Specific problem to fix

**Learning Velocity:**
- Current rate (insights/week)
- Comparison — This week vs last vs average
- Sparkline — 8-week trend

**Autonomy:**
- Current % autonomous decisions
- Target line (80%)
- Weekly trend — Am I escalating less?

**Skill Synergy:**
- Which skills work together well
- Usage counts — How often paired
- Gap identification — Underutilized combinations

### System Tab
**Purpose:** Infrastructure and meta-training status

**Core Files:**
- Green dot = modified today (active)
- Gray dot = stale (needs review)

**Projects:**
- Expand any project for timeline
- Blockers shown in red
- Next actions listed

**Meta-Training:**
- Portfolio of work with grades
- Patterns that work
- Checkpoints reached

**Honest Assessment:**
- Current grade (C+)
- Target grade (B-)
- Specific gaps to close
- What's working (wins)

---

## Reading Patterns

**Good signs:**
- Green indicators (↑ improving)
- Calibration > 75%
- Autonomy trending up
- Few escalations

**Warning signs:**
- Red indicators (↓ declining)
- Calibration < 60% (overconfident)
- Stuck autonomy %
- Repeated blockers

## Time to Understand

Each section designed for 3-second comprehension:
1. Read the statement
2. Check the primary metric
3. Note the insight

If it takes longer, the design failed.
