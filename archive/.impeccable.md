## Design Context

### Users
Marco Lam — 21, systems thinker, high-agency founder building an autonomous trading agent. Uses dashboard to monitor Claw's progress, validate edges, and make deployment decisions. Context: juggling school, business, and Claw — needs information that drives action, not raw data to parse.

### Brand Personality
**Thermal Predator** — Sharp, focused, dangerous precision. Not friendly or playful. Confident without being arrogant. Every element should feel like it was designed by someone who trades serious money.

Three words: Precise, Predatory, Professional

Emotional goal: Confidence in the system's capability. Urgency when intervention needed.

### Aesthetic Direction
Dark mode with amber (#E0912A) as sole accent. No gradients, no neon, no glassmorphism for decoration. Deep charcoal backgrounds (#141414) with layered depth. Typography: Geist Sans + JetBrains Mono for data.

References: Bloomberg Terminal, Linear, Figma's dark mode — but stripped to essentials.
Anti-references: Crypto dashboards with neon glows, generic admin templates, "AI slop" with rounded corners everywhere.

### Design Principles
1. **Every card answers "so what?"** — Information must drive action or insight
2. **Motion serves purpose** — Guide attention, confirm actions, never decorate
3. **3-second comprehension** — Hierarchy so clear the insight is instant
4. **Depth through layers** — Elevation system, not flat cards
5. **Thermal language** — Amber heat on cold charcoal, predator focus

### Technical Constraints
- Single HTML file, no external dependencies
- SVG for charts, no D3.js
- CSS animations only, lightweight
- Mobile responsive (320px+)
